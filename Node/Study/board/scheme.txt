    - 3 계층 구조란, 웹 프레임워크에서 주로 사용하는 MVC 패턴을 적용하기에도 좋다.
        3 계층 아키텍처 ( 컨트롤러 - 서비스 - 데이터 액세스 )
        익스프레스에서 3 계층 아키텍처 ( 익스프레스 라우터 - 서비스 클래스 또는 함수 - 몽고디비 모듈 )
